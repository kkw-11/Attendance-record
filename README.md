# Attendance-record

## Introduction
- 출근 기록 관리 프로그램 개발
- Arduino Client 카드리더기에 카드 인식을 하면 등록된 카드라면 Raspberry Pi Server로 카드 정보 전송, 출근 시간 DB에 저장, 등록되지 않은 카드라면 Buzzer를 울림

## Tech Stack
 ### Client
  - Arduino, RFID(Radio-Frequency Identification), LED, Buzzer, Resitance
 ### Server
  - C, Mysql, Raspberry Pi
